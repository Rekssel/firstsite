2. Внутренние			в самом документе
3. Инлайновые стили		прямо в теге через атрибут style=""
4. Импорт-стили			если ты этим воспользуешься, то у тебя будет сыпь,

h1 - селектор
{
	color: - свойство
	red; - значение свойства
}

.%имя_класса% {
	установки для всех объектов этого класса
}

#%имя_идентификатора% {
	установки для элемента с данным идентификатором
}

.%имя_класса%#%имя_идентификатора% {
	установки для объекта этого класса с данным идентификатором
}

.%имя_класса%<.%имя_класса%> {
	установки для объекта этих классов класса
}

!задавать стили идентификаторам не рекомендутся!

селектор наследников

<%имя_элемента> <.%имя_класса%> %имя_элемента_X% {
	установки для объекта %имя_элемента_X% , у которого родители (<%имя_элемента> <.%имя_класса%>) расположены в обратной последовательности
}
